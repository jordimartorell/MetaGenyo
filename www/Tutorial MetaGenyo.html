MetaGenyo: Simplifying your meta-analysis

TUTORIAL

Introduction

MetaGenyo is a simple, ready-to-use software which has been designed to perform meta-analysis of genetic association studies.

Get your data

The first step to do a meta-analysis is, of course, to obtain the data. You must search in bibliography genetic association studies related with the disease you are interested in. You will find a lot of SNPs associated to that disease. You must choose one of these SNPs and save the data for that SNP of multiple studies. For saving and preparing your data you can use any suitable program (Microsoft Excel, LibreOffice Calc...).

Prepare your data

MetaGenyo requires a specific format of data to work. The first n columns of your file can contain any type of data of the different studies. N is defined by yourself, but must be a number between 1 and 10. The following columns must be exactly like this:

AA cases Aa cases aa cases AA controls Aa controls aa controls


















The meaning of each column are the following:

  • AA cases: The number of subjects with the disease (cases) that are homozygote for the A variant.
  • Aa cases: The number of cases that are heterozygote for the A and a variants.
  • aa cases: The number of subject case) that are homozygote for the a variant.
  • AA/Aa/aa controls: The same as the previous columns, but for the number of subjects with no disease (controls).
IMPORTANT: A and a variants are merely examples with no more significance. That means that A and a can correspond to any variant that you want to study.

In conclusion, your data should be something like this (remember that previous columns to "AA cases" column are choosen by yourself):

Author Country Ethnicity AA cases Aa cases aa cases AA controls Aa controls aa controls



























You can write the column names that you want. The important is that data must be in the correct order.
